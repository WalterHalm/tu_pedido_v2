🍕 Tu Pedido v2

Sistema Completo de Gestión de Pedidos para Restaurantes

Tu Pedido v2 es un sistema completo de gestión de pedidos diseñado específicamente para restaurantes de comida rápida. Unifica pedidos del eCommerce y PoS en un dashboard interactivo con notificaciones en tiempo real.

🎯Dashboard Interactivo

  • Vista Kanban con drag & drop
  • Estados en tiempo real
  • Notificaciones sonoras automáticas
  • Seguimiento de tiempos

🔔Notificaciones Unificadas

  • 🌐 Pedidos web nuevos
  • 🚚 Pedidos listos para enviar
  • 📍 Pedidos listos para retirar
  • Modales con acciones rápidas

🔄Automatización

  • Detección automática delivery/pickup
  • Desactivación automática de alertas
  • Formateo inteligente de nombres
  • Sincronización PoS-eCommerce

🚀 Características Principales

  • Integración Completa: eCommerce + PoS unificados en un solo dashboard
  • Tiempo Real: Actualizaciones automáticas cada 30 segundos
  • APIs Completas: Para integración con sitios web y seguimiento de clientes
  • Responsive: Funciona perfectamente en desktop y tablet
  • Notificaciones Inteligentes: Sistema unificado con 3 tipos de alertas
  • Gestión de Estados: 7 estados de pedido con transiciones automáticas

📋 Estados del Pedido

Nuevo → Aceptado → En Preparación → Terminado → Despachado → Entregado

Cada estado tiene tiempos de seguimiento automático y notificaciones correspondientes.

🔧 Compatibilidad

  • Odoo: 18.0 Community
  • Navegadores: Chrome, Firefox, Safari, Edge
  • Dispositivos: Desktop, Tablet (responsive)
  • Dependencias: sale, website_sale, portal, point_of_sale, pos_restaurant
